WICHTIG fuer E2E-Fehler "strict mode violation: getByText(...) resolved to N elements":
  Das bedeutet, der Text erscheint in MEHREREN DOM-Elementen gleichzeitig.
  Fix: Stelle sicher dass der gesuchte Text nur in EINEM Element vorkommt.
  Oft Ursache: Ein Badge-Text erscheint auch im Ticket-Titel oder in einem anderen Bereich.
  Loesung: Nutze aria-hidden="true" auf dem Element das den Text NICHT haben soll,
  oder aendere das Rendering so dass der Badge-Text eindeutig ist.
  HINWEIS: aria-hidden="true" auf einem Element versteckt es vor Playwright getByText().
